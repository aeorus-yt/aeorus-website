How I do things:

I made some music using FL Studio Producer edition on my main system using Linux KVM and QEMU to virtualize an instance of Windows 10.

I make content using OBS, a powerful video recording tool, and edit my videos in Davinci Resolve. This ensures an efficient workflow that allows me to make videos for you.
